## 🛠️ Instalação

```bash
# Clone o repositório
git clone https://github.com/L34NDR0-DEV/uby-agendamentos.git

# Entre no diretório
cd uby-agendamentos

# Instale as dependências
npm install

# Execute em modo desenvolvimento (com WebSocket)
npm run dev-full

# Ou execute apenas a aplicação (modo offline)
npm run dev

# Build para produção
npm run build
```

---
